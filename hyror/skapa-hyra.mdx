# Skapa hyra

En hyra i Tenovo representerar en faktura för en specifik period kopplad till ett hyreskontrakt. Du kan lägga till flera rader på samma hyra för att fakturera både hyra, parkering och andra tillägg.

## Steg för steg

1. Gå till **Hyror** i huvudmenyn.
2. Klicka på **Skapa hyra** uppe till höger.
3. Välj det **hyreskontrakt** som hyran ska kopplas till.
4. Fyll i perioden:
   - **Fakturaperiodens start- och slutdatum**
   - **Förfallodatum**
5. Lägg till rader:
   - **Beskrivning** – exempelvis *Hyra november*, *Parkering*
   - **Belopp**
   - **Momssats** – justera om annan moms gäller
6. Klicka på **Spara utkast**.

Hyra sparas som utkast tills du väljer att skicka den.

## Skicka hyran

När utkastet är klart kan du ändra status till **Skickad** för att markera att hyran har skickats till hyresgästen. I framtida versioner av Tenovo kommer det även att gå att skicka hyran direkt via e-post.

<Tip>
Om du skapar hyror regelbundet kan du utgå från ett aktivt kontrakt för att slippa fylla i hyresbelopp manuellt varje gång.
</Tip>

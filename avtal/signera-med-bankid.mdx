# Signera med BankID

Tenovo stödjer elektronisk signering av hyreskontrakt med BankID. Det innebär att både du som hyresvärd och hyresgästen kan signera kontraktet digitalt – utan att behöva träffas fysiskt eller skriva ut papper.

## Så fungerar det

1. Du skapar ett hyreskontrakt och sparar det som utkast.
2. När du är redo klickar du på signera-knappen för att initiera signering.
3. Beroende på vem som ska signera först väljer du antingen:
   - **Signera som hyresvärd** – du signerar först
   - **Skicka för signering till hyresgäst** – hyresgästen signerar först
4. Den andra parten signeras därefter, och kontraktet markeras som **Aktivt**.

## Signera som hyresvärd

1. Öppna kontraktet från listan eller Kanban-tavlan.
2. Klicka på **pennikonen** (*Signera som hyresvärd*).
3. Bekräfta signeringen med ditt BankID på den enhet du väljer.

## Skicka för signering till hyresgäst

1. Öppna kontraktet.
2. Klicka på **dokumentikonen** (*Skicka för signering till hyresgäst*).
3. Hyresgästen får en länk för att signera med sitt BankID.
4. När hyresgästen har signerat kan du signera som hyresvärd om du inte redan gjort det.

<Warning>
Ett kontrakt som väntar på signering kan inte redigeras. Om du behöver ändra något måste du avbryta signeringsprocessen och spara om utkastet.
</Warning>

## Efter signering

När båda parter har signerat:

- Kontraktets status ändras till **Aktivt**
- En signerad PDF genereras
- Både du och hyresgästen kan ladda ner avtalet från respektive portal
- Automatiska händelser kan skapas i kalendern, beroende på dina inställningar
